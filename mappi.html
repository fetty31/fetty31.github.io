MaPPI

Lightweight Model Predictive Path Integral controller for autonomous navigation. MaPPI is designed for real-time obstacle avoidance, smooth trajectory generation, and supports kinematic bicycle models for vehicle-like robots.

View on GitHub

Overview

MaPPI implements a lightweight MPPI controller for autonomous robots, enabling kinematically feasible path planning and trajectory following. Unlike the Nav2 MPPI controller, MaPPI directly supports vehicle-like robots through a kinematic bicycle model and provides a standalone C++ library for use outside ROS.

Distinctive Features

  • Kinematic Bicycle Model: suitable for car-like robots (steering + velocity).
  • Standalone C++ library: portable, works independently of ROS.
  • ROS Noetic support: C++ plugin for ROS 1 `move_base` via `nav_core::BaseLocalPlanner`.
  • MotionModel-based trajectory generation ensures realistic robot motion compliance.

Technology Stack

  • C++14 for core controller and motion model.
  • ROS Noetic and ROS2 Humble plugins for integration with existing navigation stacks.
  • Eigen and xtensor for efficient matrix and array operations.

Getting Started

Clone the repository and follow the README instructions to integrate MaPPI into your ROS-based navigation system, or use it standalone for custom experiments.

Results

MaPPI generates smooth, feasible paths with real-time obstacle avoidance. Its kinematic support and standalone design make it versatile for embedded platforms and vehicle-like robots, offering a clear advantage over the Nav2 MPPI controller in ROS 1 setups.
