Tailored MPC

Non-linear curvature MPC developed for BCN eMotorsport Formula Student team. Focused on lateral control for the CAT15x prototype.

View on GitHub

Overview

Tailored MPC is a non-linear curvature Model Predictive Controller developed for the CAT15x Formula Student car. It implements a high-performance lateral controller that was actively deployed during the 2023 competition season, providing precise steering and optimal vehicle handling on the track.

This software was shared as part of my Bachelor’s thesis and was actively deployed during the 2023 competition season, contributing to the vehicle’s performance on the track.

Features

  • Curvature-based non-linear MPC for robust lateral control at high speeds.
  • Optimized for real-time execution on embedded vehicle hardware.
  • Fully validated on the CAT15x Formula Student car during competition.
  • Supports fast tuning for different track conditions and vehicle dynamics.

Technology Stack

  • C++ for core real-time control algorithms.
  • ROS Noetic for vehicle integration and data streaming.
  • Eigen3 for efficient matrix operations.
  • FORCESPRO solver for NMPC optimization.

Getting Started

Clone the repository and follow the README instructions to run the controller in simulation or on a Formula Student vehicle.

Results

The lateral controller delivers smooth and precise steering, enabling competitive lap times and safe handling on the track. Its real-time performance and reliability were demonstrated during the 2023 Formula Student competition season.
