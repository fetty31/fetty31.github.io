Fast-LIMO

Lightweight and efficient LiDAR-inertial odometry system designed for real-time robotic applications.

View on GitHub

Overview

Fast-LIMO is a fast, lightweight and tightly coupled LiDAR-inertial odometry (LIMO) system optimized for real-time robotic navigation. It integrates 3D LiDAR scans and IMU data to provide accurate pose estimation while maintaining high efficiency.

Features

  • Lightweight and optimized for real-time performance.
  • Fuses LiDAR and inertial measurements for accurate odometry.
  • Robust to motion distortions and noisy sensor data.
  • Easy integration with ROS-based robotics systems.

Technology Stack

  • C++ for core algorithms.
  • ROS/ROS2 for robot integration.
  • Eigen for matrix operations.
  • PCL (Point Cloud Library) for LiDAR processing.

Getting Started

Clone the repository and follow the instructions in the README to build and run Fast-LIMO on your robot or dataset.

Results

Fast-LIMO provides smooth and accurate odometry for mobile robots in indoor and outdoor environments. Its lightweight design ensures low latency and minimal CPU usage, making it suitable for embedded systems. Although I initially developed it for autonomous racing applications, it has been successfully tested in various robotic platforms, demonstrating its versatility and robustness in real-world scenarios.
