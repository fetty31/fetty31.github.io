LieOdyssey

Lightweight, research-friendly toolkit for IMU preintegration and state estimation on Lie groups.

View on GitHub

Overview

LieOdyssey provides efficient preintegration routines and filters (observers) on common Lie Groups, enabling robust navigation and state estimation for robotics, drones and autonomous systems. Built on the mathematical foundation of Lie Theory, it simplifies working with IMU data and inertial navigation pipelines.

Features

  • Preintegration on any Group from Lie++ or ManIf.
  • Lie algebra tools — exponential/logarithm maps, Jacobians, Adjoints.
  • Ready-to-deploy observers for SLAM/INS estimation (ESEKF, InvEKF).
  • Lightweight, modular, easy integration with factor graphs (GTSAM, Ceres).

Technology Stack

  • C++ core library.
  • Lie++ or ManIf for Lie group operations.
  • ROS/ROS2 integration supported.

Getting Started

Clone the repository and build the library as described in the README. Optionally, build tests with CMake or integrate it in a ROS workspace.

Background

LieOdyssey leverages Lie group theory for geometric consistency, fast optimization and accurate uncertainty propagation in inertial navigation systems. Preintegration allows summarizing high-rate IMU data into compact measurements, reducing computation and improving filter robustness. LieOdyssey is my personal playground for building Lie group observers from the ground up — learning, experimenting and having fun with state estimation.
