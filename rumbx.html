rumbX

MATLAB‑based optimal control framework to compute time‑optimal racing trajectories using a dynamic bicycle model.

View on GitHub

Overview

rumbX is an optimal control solver written in MATLAB for racing trajectory optimization. It formulates the problem using a dynamic bicycle model with simplified tire forces and solves for a time‑optimal racing line along a track, respecting vehicle dynamics, track boundaries, and control limits. This project couples vehicle physics with nonlinear programming to compute the fastest feasible trajectory.

Features

  • Processes midline track coordinates from CSV formats.
  • 7‑state dynamic bicycle model with simplified Pacejka tire force approximation.
  • 2 control inputs: steering rate and acceleration rate.
  • Spatial OCP formulation with state/control bounds and track limits.
  • Generates optimal state and control profiles along the trajectory.

Technology Stack

  • MATLAB (primary development environment).
  • CasADi (symbolic framework) for modeling and interfacing with IPOPT.
  • IPOPT nonlinear solver for optimization.

Getting Started

Clone the repository and ensure you have MATLAB R2021+ installed along with CasADi 3.6 and IPOPT. Follow the instructions in the README to load track data (CSV), set problem parameters, and solve the optimal control problem using the provided scripts.

How It Works

  • The OCP is formulated in the spatial domain using a dynamic model and cost function that maximizes progress while penalizing slip, track deviation, and control rates.
  • Constraints include nonlinear vehicle dynamics, track boundaries, state/control limits, force ellipse coupling, and periodicity (start = end).
  • Upon solving, rumbX outputs optimal trajectories and control profiles for subsequent use in simulation or control benchmarking.

Results

rumbX includes example results for well‑known tracks such as Silverstone and Catalunya, demonstrating its ability to compute physically consistent, time‑optimal racing trajectories that respect model constraints.
