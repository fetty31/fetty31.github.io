Cruise Control

Adaptive cruise controller following a GG‑based velocity profile for the BCN eMotorsport Formula Student autonomous vehicle.

View on GitHub

Overview

This project implements a longitudinal adaptive cruise controller tailored to the BCN eMotorsport Formula Student autonomous vehicle. It follows a GG‑based desired velocity profile, regulating the vehicle’s speed to maintain smooth and controlled longitudinal motion. The controller is integrated within ROS Noetic and is specifically designed for the CAT15x autonomous car.

This software was shared as part of my Bachelor’s thesis and was actively deployed during the 2023 competition season, contributing to the vehicle’s performance on the track.

Features

  • Adaptive cruise controller designed for autonomous racing platforms.
  • Follows a GG‑based velocity profile for smooth acceleration and deceleration.
  • Integrated as part of a ROS‑based autonomous control stack.
  • Modular architecture for longitudinal control tuning.

Technology Stack

  • C++ for core controller logic.
  • ROS Noetic for robot integration and communication.
  • Eigen3 for efficient linear algebra computations.
  • ROS messages and custom message packages for control interfacing.

Getting Started

Clone the repository and build it within your ROS Noetic workspace. Ensure dependencies such as Eigen3 and the associated custom ROS message packages are properly configured (note: some ROS message packages are proprietary to the BCN eMotorsport team).

Approach

The controller’s architecture is based on established cruise control design principles, using feedback to track a desired velocity profile while minimizing throttle oscillations and preserving stability during speed changes. An anti‑windup strategy is included to prevent integrator build‑up during actuator saturation.

Usage

Once built and sourced, run the cruise control node within your ROS environment as part of the vehicle’s autonomous systems stack. Adjust velocity profile parameters as required for your test scenario or track configuration.
